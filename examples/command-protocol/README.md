# Command Protocol Example

Demonstrates Vexil for designing RPC-style command protocols:

- **Unions** — `Command` with typed variants (`Ping`, `Auth`, `Query`), length-prefixed for forward compatibility
- **Flags** — `Permissions` bitset with named constants and set operations
- **Enums** — `Status` with compact wire representation
- **Limits** — `@limit` annotations enforce max sizes at encode time

## Run

    cargo run

## Files

- `commands.vexil` — the protocol schema
- `src/generated.rs` — Rust code generated by `vexilc codegen`
- `src/main.rs` — construct, encode, and decode various command types
