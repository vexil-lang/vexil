# Cross-Language Example: Sensor Telemetry

Demonstrates Rust <-> Node.js <-> Go interop using the Vexil wire format.

A Rust "device" encodes sensor readings -> binary file -> Node "dashboard" and Go "gateway" decode them.
The Node dashboard and Go gateway each encode a config command -> binary file -> Rust device decodes it.

All three sides use code generated by `vexilc` from the same schema, ensuring wire compatibility.

## Schema

See `schema/telemetry.vexil` -- defines `SensorReading` (with sub-byte fields, enums, optionals) and `DeviceConfig`.

## Running

### 1. Rust encodes sensor readings

```bash
cd examples/cross-language/rust-device
cargo run
```

This writes 3 sensor readings to `data/readings.bin`.

### 2. Node decodes readings, writes config

```bash
cd examples/cross-language/node-dashboard
npm install
npm start
```

This reads `data/readings.bin`, prints the decoded readings, then writes a `DeviceConfig` to `data/config.bin`.

### 3. Go decodes readings, writes config

```bash
cd examples/cross-language/go-gateway
go run .
```

This reads `data/readings.bin` (from Rust), prints decoded readings, then writes a `DeviceConfig` to `data/config_from_go.bin`. If `data/config.bin` exists (from Node), it decodes that too.

### 4. Rust reads config from Node

```bash
cd examples/cross-language/rust-device
cargo run
```

Run again -- it will encode new readings AND decode the config from `data/config.bin`.

## Regenerating code

If you modify the schema:

```bash
cargo run -p vexilc -- codegen examples/cross-language/schema/telemetry.vexil --target rust > examples/cross-language/rust-device/src/generated.rs
cargo run -p vexilc -- codegen examples/cross-language/schema/telemetry.vexil --target typescript > examples/cross-language/node-dashboard/src/generated.ts
cargo run -p vexilc -- codegen examples/cross-language/schema/telemetry.vexil --target go > examples/cross-language/go-gateway/generated.go
```

For the Go gateway, you can also use `go generate`:

```bash
cd examples/cross-language/go-gateway
go generate ./...
```
