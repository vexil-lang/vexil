## Orchestration Loop Used

| Phase | Agent | Model | Mechanism |
|-------|-------|-------|-----------|
| Planning | OpenCode | Kimi K2.5 Turbo | `opencode run --model` |
| Review | Hermes subagent | MiniMax 2.7 | `delegate_task` |
| Implement (most) | OpenCode | Kimi K2.5 Turbo | `opencode run --model` |
| Error messages + clippy fixes | Hermes (manual) | mimo-v2-pro | Direct patch |

### Lessons Learned
- OpenCode with focused prompts (10-15 lines) succeeds reliably within 120s
- Long prompts (30+ lines) with full pipeline descriptions tend to timeout
- `delegate_task` model override doesn't work with custom providers (use OpenCode)
- OpenCode sessions that "timed out" often completed the work but timed out on output
